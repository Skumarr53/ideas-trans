# Project Overview

Sentiment Analysis Agent is a modular, production-grade platform for running sentiment analysis on financial call transcripts. It provides:

* Batch inference and fine-tuning workflows
* Spark/Databricks integration
* Streamlit UI and REST API support
* Pluggable model architecture (NLI, Gemma, future models)
* MLflow experiment tracking
* Secure, scalable, and container-ready setup
